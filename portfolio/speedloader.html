Speedloader

Speedloader is a content delivery system I created for Mosaic. This system was created so clients could easily login and download media elements that Mosaic had created for them. This required a way for Mosaic to manage their media files and which clients had access to them.

I was the sole developer on this project and developed it from the ground up. This system was built with jQuery, MySQL and PHP.
